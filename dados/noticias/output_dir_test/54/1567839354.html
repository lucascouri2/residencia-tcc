Coronavirus dans le Lot. Le marché de Lalbenque autorisé

Un nouveau marché a été autorisé ce vendredi 3 avril 2020 dans le Lot en plaine crise du coronavirus : celui de Lalbenque.

Par Rédaction Cahors Publié le 3 Avr 20 à 16:19 
Actu Lot
Voir mon actu

Le Préfet du Lot vient d’autoriser la tenue d’un nouveau marché dans le Lot ce vendredi 3 avril 2020 : celui de Lalbenque qui a lieu le samedi matin.

C’est donc le 16e marché à être autorisé par arrêté dérogatoire dans le département du Lot. 

Conformément aux annonces du Premier ministre du 23 mars 2020, les marchés couverts et de plein air du département ont été fermés. Néanmoins, dans des cas exceptionnels, il est permis au préfet, sur avis des maires, de déroger à cette interdiction, notamment dans les communes non pourvues de commerces de proximité, et sous réserve du respect des règles de distanciation sociale.

Liste des marchés autorisés dans le Lot

  • Beauregard, les mardis en fin d’après midi
  • Cazals, les dimanches matin
  • Concots, les dimanches matin
  • Gourdon, les samedis matin
  • Gramat, les mardis et samedis matin
  • Lalbenque, les samedis matin
  • Luzech, les mercredis matin
  • Montvalent, les samedis matin
  • Puy l’évêque, les mardis matin
  • Prudhomat, les mardis après-midi
  • Saint-Germain-du- Bel-air, les vendredis matin
  • St Géry-Vers, les dimanches matin
  • Saint-Céré, les dimanches matin
  • Souillac, les vendredis matin
  • Théminettes, les vendredis en fin d’après-midi
  • Vayrac, les jeudis et samedis matins
